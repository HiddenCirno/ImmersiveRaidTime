# 星火计划·沉浸式战局时间

让你的游戏时间和现实同步。

**它能做什么？** 

- 让你的战局时间和现实完全同步，并且可以自由调整时区和开关夏令时。

- 启用沉浸式战局，让右上角的战局计时器显示现实时间，战局时间将变为无限并禁用迷失功能。

- 对游戏的QOL功能进行二次改进，单击O键时，时间显示器还会显示现实时间（这个功能不受自定义时区影响）。

**如何使用？** 

- 在右侧的Release下载发布版本，解压后得到BepInEx文件夹，直接覆盖于游戏根目录即可。

**兼容性如何？**

- **与Fika的兼容性：** 完全兼容，我在实际测试中没有遇到任何问题。

- **与Fika.Co-op的兼容性：** 不确定，尚未测试，应该会以主机为准。

- **与Fika.Headless的兼容性：** 不确定，尚未测试，但应该会以Headless为准，同上。
